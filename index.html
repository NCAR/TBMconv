NCAR Terabit Memory System (TBM) Archive Conversion Utility & Documentation

View on GitHub

Getting the source

$ git clone https://github.com/NCAR/TBMconv.git

Compiling

tbmconv and tbmexplore have no external dependencies.

$ cd TBMconv
$ make

Documentation

A PDF document describing the TBM format is available: tbm.pdf
